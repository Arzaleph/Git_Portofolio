# Portofolio
*JUDUL PROYEK*
Website Portfolio Arza Restu Arjuna

*DESKRIPSI PROYEK*
Proyek ini merupakan implementasi website portfolio pribadi. Website ini dikembangkan sebagai web statis, yang berarti hanya menggunakan bahasa sisi klien seperti HTML dan CSS tanpa memerlukan pengolahan data di server. Proyek ini dibuat untuk memenuhi Tugas Akhir Praktikum Pemrograman Web (PEMWEB).
Fokus utama dari website statis ini adalah untuk menampilkan informasi secara cepat dan efisien. Website ini menyajikan:
1.	Profil singkat dan latar belakang akademis [Sesuaikan dengan isi portfolio Anda].
2.	Daftar keahlian teknis (Skills) yang relevan, seperti HTML, CSS, dan penggunaan Git [Sesuaikan dengan isi portfolio Anda].
3.	Koleksi proyek-proyek yang telah diselesaikan.
4.	Informasi kontak.
Seluruh proses pengembangan, termasuk tahap branching dan merging, dicatat dan dikelola menggunakan Git Distributed Version Control System.

*INSTALASI DAN PENGGUNAAN*
Karena ini adalah web statis, tidak ada langkah instalasi server yang diperlukan. Anda hanya perlu mendapatkan salinan file proyek ini.
Langkah 1: Mengunduh Proyek (Kloning Repository)
Untuk mendapatkan semua file proyek, gunakan perintah Git Clone:
git clone https://github.com/Arzaleph/Git_Portofolio
Pindah ke folder proyek:
cd Git_Portofolio
Langkah 2: Menjalankan Website
Buka file index.html yang ada di dalam folder proyek menggunakan browser web apa pun seperti Chrome, Brave, Firefox, Edge, dll.

*STRUKTUR DAN TEKNOLOGI*
Website ini murni dikembangkan menggunakan HTML dan CSS.
Git Workflow: Branch utama yang digunakan adalah master, dan branch eksperimen yaitu experiment-styling digunakan untuk fitur dan perubahan styling sebelum di-merge ke branch utama.
Folder hasilTa2 berisi bukti pengerjaan, seperti screenshot log Git.
